# Linux and Unix MySQL commands

[reference](http://www.computerhope.com/unix/mysql.htm)

* link to MySQL   
`mysql -h [ip] -u [account] -p[password]`   
__NOTE__ There is no space between `-p`and it's password   

## In MySQL console

* help   
`\h`   

* link to database   
`\r [database's name]`   

* Send command to MySQL   
`[MySQL command]`   
`\G`
